Course and Research Projects

    Functional Data Analysis for Zillow Home Value Index (ZHVI) Data

    Nov 2019 - Dec 2019 (Functional Data Analysis Course Project)  

  • Use functional data methods to study the ZHVI data (data visualization, outlier detection, data inference, data clustering, exploring the underlying correlation between two random functional objects).

    Automatic Traits Extraction and Fitting for Field High-throughput Phenotyping Systems

    Jun 2019 - Present (RA project)  

  • Propose algorithm to accurately segment plants from field images using NN, and introduce a novel procedure to separate the plants from the front and back rows based on CNN.
  • Develop robust procedure for automatic plant height measurement and plant growth curve fitting.

    An RKHS Approach for Variable Selection in High Dimensional Functional Linear Models

    Feb 2019 - Present (Ph.D thesis project)  

  • Propose an elastic-net type estimator for functional linear regression with scalar responses and functional predictors under a reproducing kernel Hilbert space framework.
  • Establish precise conditions on the problem dimension (number of observations, total number of features, number of true features) and the scope and ratio of L1 and L2 penalty that are sufficient for variable selection consistency.

    A Hierarchical Spatial Finlay-Wilkinson Model for Analysis of Multi-Environment Field Trials

    Jan 2018 - Present (Ph.D thesis project)   [See project]

  • Build a Bayesian hierarchical model for understanding and predicting crop performance across environments by integrating massive data from different sources (i.e. genomic, environmental, and within-field spatial data).
  • Propose methodology for simultaneously modeling spatial correlation in multi-environment trials and design fast computation algorithms for simulating high-dimensional spatial effects in MCMC procedures.
